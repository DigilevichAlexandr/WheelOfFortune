# Колесо фортуны (.NET 8)

Веб-приложение для розыгрыша призов:
- Участники добавляют очки.
- Очки формируют доли колеса.
- При вращении случайно выбирается победитель по весу доли.
- Победитель забирает весь банк и может списывать очки на призы.

## Локальный запуск

Требования: .NET SDK 8.0+

```bash
dotnet restore
dotnet run
```

По умолчанию приложение откроется на `http://localhost:5000` (или порту из окружения).

## Деплой на Render.com

В репозитории уже есть:
- `Dockerfile` для сборки/запуска ASP.NET Core приложения.
- `render.yaml` с описанием веб-сервиса.

Шаги:
1. Загрузите репозиторий в GitHub.
2. В Render выберите **New +** → **Blueprint**.
3. Подключите репозиторий.
4. Render автоматически прочитает `render.yaml` и создаст сервис.

## API

- `GET /api/state` — текущее состояние игры.
- `POST /api/participants` — добавить участника.
  - body: `{ "name": "Иван", "points": 120 }`
- `POST /api/spin` — крутить колесо.
- `POST /api/redeem` — списать очки победителя на приз.
  - body: `{ "winnerId": "...", "prizeCost": 50 }`
